# Runtime Profiles

Runtime Profile 是 LumiForge 的核心部署单元之一。

第一版内置：

- `esp-claw-compatible.json`

该 Profile 用于复刻 ESP-Claw 的能力面：

- Agent Core
- Capability Runtime
- Event Router
- LLM + Lua Engine
- Local Memory
- Skills Manager
- MCP Server / Client
- IM Platform
- Web Console
- Runtime File System
- Scheduler
- File Manager

后续会增加：

- `lumiforge-esp32s3.json`
- `xiaozhi-compatible.json`
- `linux-sbc-runtime.json`
