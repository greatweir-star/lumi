# Agent Templates

Agent Template 不能只是人设模板。

在 LumiForge 中，Agent Template 后续必须包含：

- persona
- default session policy
- required capabilities
- required runtime modules
- skills
- lua scripts
- router rules
- memory seed
- mcp tool bindings
- web console defaults

当前 `agents.json` 仍是轻量版本，后续需要升级到 Runtime-aware Template。
