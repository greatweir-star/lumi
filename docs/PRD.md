# 《灵机工坊》产品需求文档 PRD v0.1

## 1. 产品定位

灵机工坊是一站式 AI 智能硬件开发平台，面向 AI 硬件创业者、独立开发者、方案商和内容型硬件产品团队，提供从硬件选型、Agent 配置、一键烧录、在线调试、OTA 到模板发布的完整链路。

一句话：让开发者不用从 0 搭嵌入式系统，也能快速把一个 AI 想法灌进真实硬件。

## 2. 目标用户

- AI 硬件创业者：需要快速做原型和验证商业闭环。
- 独立开发者：希望低门槛制作 AI 吧唧、桌宠、语音助手等设备。
- 方案商：需要更快交付客户项目和沉淀模板。
- 内容/IP 团队：希望把角色、人格、内容做成可交互硬件周边。

## 3. MVP 范围

第一阶段聚焦 AI 陪伴类智能硬件：

- AI 陪伴吧唧
- AI 桌面宠物
- AI 语音助手
- AI 拍照识别助手
- AI 情绪灯

首批硬件：

- ESP32-S3 屏幕开发板
- M5Stack CoreS3
- 小智 ESP32 语音硬件
- Seeed XIAO ESP32S3 Sense

## 4. 核心流程

1. 选择应用场景。
2. 平台推荐硬件。
3. 选择 Agent 模板。
4. 配置人设、模型、语音、记忆、技能和触发规则。
5. 浏览器连接设备。
6. 一键烧录固件和 Agent 配置。
7. 进入在线调试台。
8. 保存项目，后续支持 OTA、模板发布和批量量产。

## 5. 核心模块

- 硬件中心：设备能力、芯片、外设、成熟度、推荐场景。
- 模板市场：Agent 应用模板、技能包、UI 表情、固件 Profile。
- Agent Studio：配置人设、记忆、技能、MCP 工具和部署参数。
- Web 烧录器：Web Serial 设备连接、一键刷写、配置灌装。
- 设备调试台：串口日志、联网状态、外设自检、部署进度。
- API 服务：设备、模板、项目、部署记录、推荐算法。

## 6. MVP 验收标准

- 可以看到支持硬件列表和 Agent 模板列表。
- 可以进入 Agent Studio 原型页。
- API 可以返回 devices、templates 和 recommendations。
- 可以创建项目并生成 Agent Config。
- 部署接口可以模拟烧录进度。
- 代码结构支持后续接入真实 Web Serial 和 esptool-js。
